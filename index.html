KADAI

Some of the functionalities provided by KADAI are prioritization, distribution of tasks, monitoring the current progress and customization of the task attributes. KADAI can also help to centralize task management within larger organizations with multiple task management mechanisms. KADAI is available as a Java library and as a REST API with lots of customization options for your task management system. It can be operated as standalone or be used to build integrated task management into your own application as well.

Open Source Software for Task Management

Workbasket

A Workbasket is a list of Tasks. Each Task is placed into exactly one Workbasket. You can specify access rights and other attributes of a Workbasket.

Routing

KADAI can route Tasks into different Workbaskets. The routing is performed automatically according to customized rules.

Monitoring

KADAI visualizes the current state of work. This includes Tasks, their states, their due dates etc.. For example, you can view the number of due Tasks plotted by time.

Prioritization

In KADAI, tasks can be prioritized manually or automatically. They can also be reprioritized dynamically during their lifetime.
